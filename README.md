# Diseño Blanco

Sitio web para empresa de arquitectura. Base tomada de la plantilla **"Estudio Técnico"** (plantilla-3) del repositorio [plantillas-diseno-blanco](https://github.com/Davidcr33p/plantillas-diseno-blanco).

## Estructura

- `index.html` — estructura de la página (secciones vacías, se llenan por JS).
- `css/styles.css` — skin visual de la plantilla (tema oscuro, tipografía Archivo + JetBrains Mono).
- `js/data.js` — **todo el contenido del sitio vive aquí** (nombre, textos, servicios, proyectos, contacto). Editar solo este archivo para cambiar textos/imágenes.
- `js/main.js` — lógica que renderiza el sitio a partir de `data.js` (nav, hero, galería con filtros y lightbox, etc.).
- `assets/` — fuentes e imágenes. Las imágenes actuales son **placeholders temporales** (ver `assets/images/CREDITS.md` para licencias) hasta recibir el material real del cliente.
